# upGrad Virtual Event Management System (EMS)

## Project Overview
The upGrad Virtual Event Management System (EMS) is a web-based application developed using **HTML5, CSS3, Bootstrap 5, and JavaScript**.  
It allows users to explore and register for virtual events, while the admin can manage events using **CRUD operations** with **IndexedDB**.

---

##  Features

### Participant
- View upcoming events on Home page
- Register for an event
- Contact the organization using Contact Us page

### Admin
- Login with JavaScript authentication
- View all events
- Add new events
- Search events by ID, Name, or Category
- Edit/Update events
- Delete events
- Logout securely

## How to Run the Project
1. Open the project folder in VS Code.
2. Open index.html in browser using Live Server or double-click.
3. Use the Login page for Admin access.
4. After login, manage events from Events page.
